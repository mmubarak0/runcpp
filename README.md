# runcpp
a simple program to run cpp program for linux 


**how to use**\
[1] clone this repo \
[2] chmod +x runcpp
[3] copy runcpp file were you put your code (you can copy runcpp file to /bin folder to excutable from any where)\
[4] then type runcpp as below \
